# Roadmap — GAS 중심 MVP

## MVP 범위
1. concept 브리핑 자동 게시
2. trend 브리핑 자동 게시
3. 게시 로그와 검증 규칙 정리

## Week 1 — concept 자동 게시
- concept markdown 구조 정리
- manifest 기반 순차 게시
- Script Properties progress 저장
- GAS `runConceptDaily()` 구현 및 검증

## Week 2 — trend 자동 게시
- 최근 7일 논문 수집
- citation count 기준 상위 3편 선택
- few-shot topic tagging
- GAS `runTrendWeekly()` 구현 및 검증
- Google Sheets history 기록

## Week 3 — 후속 설계
- 퀴즈 게시 포맷 정리
- 입력 수집 방식은 현재 범위에서 제외

## 현재 후순위/제외
- Discord interaction
- 리더보드
- 실시간 제출/채점
